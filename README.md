# ML_Monitoring_And_Auto_Recovery_Agent

# MicroService and its Architecture:
This Repo would house multiple microservices interacting with HTTP to simulate the PROD enviroment and keep the architecture Clean.
| Microservice   | Path                       | venv name |
| -------------- | -------------------------- | --------- |
| Model A        | `model-services/model_a`   | `venva`   |
| Model B        | `model-services/model_b`   | `venvb`   |
| Django Backend | `control-plane/backend`    | `venvc`   |
| Agent Core     | `control-plane/agent_core` | `venvd`   |
